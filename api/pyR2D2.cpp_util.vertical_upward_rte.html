vertical_upward_rte

pyR2D2.cpp_util.vertical_upward_rte(ro: Annotated[numpy.typing.ArrayLike, numpy.float64], se: Annotated[numpy.typing.ArrayLike, numpy.float64], x: Annotated[numpy.typing.ArrayLike, numpy.float64], eos: pyR2D2.cpp_util.cpp_util.EOS) → numpy.typing.NDArray[numpy.float64]

Solve the vertical upward Radiative Transfer Equation (RTE) using the Feautrier method for given density, specific entropy, and spatial grids. :param ro_np: 3D array of density values. :type ro_np: numpy.ndarray :param se_np: 3D array of specific entropy values. :type se_np: numpy.ndarray :param x_np: 1D array of spatial grid points. :type x_np: numpy.ndarray :param eos: An instance of the EOS class for thermodynamic evaluations. :type eos: EOS

Returns:

2D array of radiative transfer results at the top boundary.

Return type:

numpy.ndarray
